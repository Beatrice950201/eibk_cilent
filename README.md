# 贝壳云笔记客户端

> 一个基于 electron-vue 开发的云储存平台

### 运行

``` bash
node -v : v12.18.2

git clone https://github.com/Beatrice950201/eibk_client.git

cd eibk_client

npm install

# 运行开发模式
npm run dev

# 打包安装文件 
npm run build

# 请自行配置src/renderer/tool/config.js

```
