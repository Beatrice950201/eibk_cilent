# 贝壳云笔记客户端

> 一个基于 electron-vue 开发的云储存平台


##### 运行

``` bash
git clone https://github.com/Beatrice950201/eibk_cilent.git

cd eibk_cilent

npm install

# 运行开发模式
npm run dev

```
